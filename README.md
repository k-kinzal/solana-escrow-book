## Getting Started

```bash
$ npm install
$ npm run build
$ open .dist/solana-escrow-book.pdf
```

PDF、WebPubの両方がビルドされるので好きな方をお使いください。

カバー画像は追加されないため、別途`articles/images/cover.png`または`articles/images/cover_dojin.png`を結合してください。
